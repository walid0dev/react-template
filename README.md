# My Niche React Template
This is a React template that I use for my projects. It is a simple template that includes the following features:
- Vite (shoutout to Evan You)
- React(duh) + React compiler
- TypeScript (because I'm not a masochist)
- Motion (for animations)
- Zod (for schema validation)
- Tailwind CSS (for styling) + a default theme from Tweak cn (you can find more themes on [Tweak.cn](https://tweakcn.com/editor/theme))
- ESLint + Prettier (for code formatting and linting)

## Getting Started
- Clone the repository
- Install dependencies with `npm install` or `yarn` or `pnpm install`
- Make it yours
```sh
rm README.md
rm -rf .git 
git init
```
- Start the development server with `npm run dev` or `yarn dev` or `pnpm dev`
- you know the drill
